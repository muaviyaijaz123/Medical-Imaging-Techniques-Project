# Medical Imaging Project: Image Processing Techniques

This repository contains the code and documentation for a project focused on implementing and evaluating various image processing techniques. The project is part of my Medical Imaging course done during my MS degree and is designed to enhance the understanding of fundamental image processing methods, particularly in the context of medical imaging.

## Project Overview

The project covers a range of image processing techniques that are critical for medical imaging applications. Key components include:

- **Thresholding Techniques**: Implementation of basic and advanced thresholding methods, including Otsu's method for automatic threshold selection.
- **Segmentation Evaluation**: Analysis of segmentation results using metrics such as Sensitivity, Specificity.
- **Edge Detection**: Application of edge detection filters such as Sobel, Scharr, Prewitt, and advanced method like Canny to highlight image boundaries.

## Directory Structure

- **src/**  
  This directory contains all Java source files implementing the image processing techniques, including:
  - **Thresholding**
  - **Edge Detection**
  - **Segmentation Evaluation**
  - **User_Interface.java**: A graphical user interface for selecting and applying techniques to grayscale images.

- **latex-template/**  
  This directory contains the LaTeX source files for the project report:
  - **project.tex**: The main LaTeX document for the project report.
  - **bibliography.bib**: BibTeX file containing references used in the report.

## How to Run the Project

To get started with the project, follow these steps:

### 1. Clone the Repository:
- git clone https://github.com/firehas123/Medizintechnik-II-Final-Project.git
- cd Medizintechnik-II-Final-Project

### 2. Set Up the Java Environment
- Ensure you have the Java Development Kit (JDK) installed.
- Use an IDE like IntelliJ IDEA or Eclipse to open the project.

### 3. Compile and Run
- Compile the Java classes.
- Run the `User_Interface.java` class to start the interface.
- Select the desired image processing technique and apply it to the image.

### 4. Access the Project Report
- Navigate to the `latex-template/` directory.
- Compile the `project.tex` file using a LaTeX editor like Overleaf or a local LaTeX distribution (e.g., TeX Live, MikTeX).
